![Flutter SDK](https://static.strivacity.com/images/flutter-sdk.png)

This SDK allows you to integrate Strivacity’s policy-driven journeys into your brand’s flutter application. The SDK uses the OAuth 2.0 PKCE flow authenticate with Strivacity.

See our [Developer Portal](https://www.strivacity.com/learn-support/developer-hub) to get started with developing with the Strivacity product.

## Getting Started

Refer to the [strivacity_flutter](./strivacity_flutter#readme) package README for detailed instructions on how to begin.
